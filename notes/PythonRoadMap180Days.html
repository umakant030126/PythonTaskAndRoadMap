⚡
Daily Coding Habit
Solve 2–3 LeetCode problems daily from Week 9 onward. Use the NeetCode 150 list as your problem bank. Quality over quantity — always explain your solution out loud.
📦
Build GitHub Portfolio
Every capstone project goes on GitHub with a README, tests, and docstrings. Recruiters check GitHub. Aim for 5 strong projects with real use-case descriptions by Month 5.
🧠
Big-O Is Non-Negotiable
For every solution you write, state the time and space complexity. Interviewers at Google and JPMC will ask this immediately. Practice until it's automatic.
🎯
Mock Interviews Weekly
From Month 4, do 1 mock interview per week with a peer or on Pramp/Interviewing.io. The ability to code while talking is a skill that requires separate practice.
📚
Resources
Books: Fluent Python, Clean Code. Sites: LeetCode, NeetCode.io, HackerRank. YT: Neetcode, Tech With Tim, Corey Schafer. Courses: Grokking System Design.

Target LeetCode Numbers
